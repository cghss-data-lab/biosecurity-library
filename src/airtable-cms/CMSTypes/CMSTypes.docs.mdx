# Types package for airtable CMS components.

```js
interface AirtableCMSData {
  nodes: [
    {
      data: {
        Name: string
        Text: string
        Image: {
          localFiles: FileNode[] &
            { childImageSharp: IGatsbyImageDataParent<IGatsbyImageData> }[]
        }
      }
    }
  ]
}
```

CMS components should share type declarations where possible, this package serves as a way to synchronize those type declarations.
